Javascript simple questions

Question-1: Deference between map(), filter(), forEach() and find() in Javascript.

Answer: map() and filter() are used to create new arrays, forEach() is used to perform an action on each element of an array, and find() is used to return the first element that meets a certain condition.

Question-2: Deference between local storage, session storage and cookies.

Answer: Cookies are ideal for storing small amounts of data that need to persist beyond the current browsing session, Local Storage is best for larger amounts of data that need to persist across multiple sessions, and Session Storage is useful for temporary data that only needs to be available during the current browsing session.

Question-3: What is the purpose of event loop in javascript?

Answer: The event loop is a key component of JavaScript's ability to handle asynchronous operations, which is essential for building responsive, interactive web applications.

what is quizTimer in javascript?

"quizTimer" in JavaScript is a script used to create a countdown timer for timed activities such as quizzes. It utilizes built-in timing functions such as setTimeout() and setInterval(), and can be customized to fit specific needs such as length of time and actions when the timer runs out.

what is chooseQuiz in javascript?

The chooseQuiz function is a key component of many interactive learning applications, enabling users to select and engage with a wide range of educational activities and quizzes.
